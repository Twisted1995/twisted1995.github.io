枚举

枚举类的好处

我们要定义常量的时候，用大写的变量通过整数定义，但是类型是int，并且仍然是变量
枚举类型通常可以看做是一种标签或者一系列常量的集合，通常用于表示某些特定的有限集合
枚举不能进行大小比较,但是可以做身份比较和 == 比较
如果继承IntEnum的话， 那么每一个枚举成员的数值都必须为数字类型
(在数据库中存取类型的时候建议使用数字来代替字符串)
枚举类内部不能有两个同名的标签的，可以通过for循环来访问枚举类中的每一个实例
‘@unique’ 装饰器可以限制枚举成员的值不重复
两个常量名称不同， 数值相同， 那么第二个就会被称作第一个的别名，而且通过for循环也不会访问到

闭包

闭包的概念： 闭包 = 函数 + 环境变量(都在一个大的函数内)
闭包的一些注意事项

  1. 局部变量不能影响外部变量
  2. 如果函数的内部重新赋值了相同变量的值， 那么就不能形成一个闭包
